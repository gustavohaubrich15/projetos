RADAR DEV

Aplicação para salvar desenvolvedores dentro do sistema web, que capta a localização atual do usuário.
No sistema mobile é possível verificar os desenvolvedores que estão a sua volta e com quais tecnologias eles trabalham.
Também ao clicar no card é disponibilizado o github deste usuário, com todas as demais informações.

Aplicação com Node Js no backend
React e React Native no frontend
